# Ҳукумати Электронӣ — E-Government Portal

Современный прототип портала электронного правительства для онлайн-доступа к государственным услугам.

## Описание

Проект позволяет гражданам:
- находить государственные услуги;
- подавать заявления онлайн;
- отслеживать статус обращений;
- пользоваться личным кабинетом;
- получать помощь через AI-чатбот;
- переключать язык интерфейса: тоҷикӣ / русский;
- использовать светлую и тёмную тему.

## Технологии

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage
- Responsive Design
- GitHub Pages

## Основные функции

- Каталог государственных услуг
- Поиск и фильтрация услуг
- AI-чатбот с моковыми ответами
- Личный кабинет
- Многошаговая форма подачи заявления
- Отслеживание статуса
- Dark / Light Mode
- TJ / RU language switcher

## Цель проекта

Цель проекта — показать, как цифровые технологии могут сделать государственные услуги более быстрыми, прозрачными и доступными для граждан.
